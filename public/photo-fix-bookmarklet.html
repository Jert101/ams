Profile Photo Fix Bookmarklet

Instructions:

  1. Drag the button below to your bookmarks bar
  2. Go to the user edit page where profile photos aren't displaying
  3. Click the bookmarklet in your bookmarks bar
  4. The profile photo should now appear

Fix Profile Photos

How It Works

This bookmarklet:

  1. Adds CSS to ensure profile photo containers are visible
  2. Finds the profile photo section on the page
  3. Ensures images are displayed correctly
  4. Fixes paths to use /profile-photos/ instead of storage paths

Troubleshooting

If the fix doesn't work:

  1. Make sure you're on the edit page when clicking the bookmarklet
  2. Try refreshing the page and clicking the bookmarklet again
  3. Check if the browser console shows any errors (press F12 to open)
