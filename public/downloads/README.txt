CKP-KofA Network Mobile App

This directory will contain the downloadable APK files for the CKP-KofA Network mobile application.

To build the APK file:

1. Navigate to the mobile-app directory
2. Follow the instructions in the README.md file to set up and build the Android app
3. Place the generated APK file in this directory with the name "ckp-kofa-app.apk"

The APK file will then be available for download from the mobile app download page.
